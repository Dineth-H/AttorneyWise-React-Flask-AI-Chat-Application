## AttorneyWise

### What Is This Project
A simple web app developed by react with react chatengine.

## How Start Project In Local

step1 :

```bash
git clone https://github.com/Dineth-H/AttorneyWise-React-Flask-AI-Chat-Application.git
```

step2 :

```bash
npm install
```

step3 :

```bash
npm run start
```

```new bash

```
